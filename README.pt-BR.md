# Fluxhook

Uma aplicação simples para montar, visualizar e enviar webhooks do Fluxer.

> 🌍 **[Read in English](README.md)** | **Leia em Português**

## Início Rápido

```bash
npm install
npm start
```

Acesse: **http://localhost:3000**

## Essencial

- Editor visual de webhook com preview em tempo real
- Suporte a embeds (título, descrição, fields, imagens, footer)
- Envio direto para URL de webhook
- Modo múltiplas mensagens (fila + reordenação)
- Internacionalização (inglês e português)
